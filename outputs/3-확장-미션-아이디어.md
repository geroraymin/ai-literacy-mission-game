# 확장 미션 아이디어 5개

현재 구조(방 추가 → NPC 대사 → 퀴즈 데이터)를 그대로 재사용해 확장할 수 있는 미션 후보입니다.
새 방 추가는 `maps.ts`에 방 하나, `missions.ts`·`quizzes.ts`·`dialogues.ts`에 항목 하나씩 늘리면 됩니다.

## 시스템 확장 아이디어 (미션 외)

- **타이머 모드**: 차시 관리용 20분 카운트다운 + 종료 시 현재 점수로 엔딩
- **학급 리더보드**: 엔딩 점수를 교사 화면에 모으기 (구글 폼 제출 링크로 간단히 시작 가능)
- **2인 협동 모드**: 한 명이 이동, 한 명이 퀴즈 결정 (역할 교대) — 코드 수정 없이 수업 규칙만으로 가능
- **아리 NPC 등장**: 미션을 깰 때마다 로비의 아리(로봇)가 점점 복구되는 연출
- **문항 난이도 2단계**: 같은 방에 기본/심화 문항 세트를 두고 시작 시 선택
