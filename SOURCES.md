# 사용할 원본 자료

## 1. 사용 원칙

- 원본 자료는 기존 위치에 그대로 둔다.
- 이 문서에는 **경로 또는 링크와 사용 이유**만 적는다.
- 반복 사용하거나 편집할 핵심 자산만 `assets\`로 선택 복사한다.
