# 프로젝트 AGENTS

이 파일은 이 프로젝트 폴더에 들어왔을 때 **가장 먼저 읽는 진입 문서**다.

## Workspace Workflow Link

- 웹 앱 작업이므로 `D:\AI_Workspace\workflows\web.md`를 구현 전에 읽는다.
- 시각 기준은 이 프로젝트의 `구현지시서.md`의 "그래픽" 섹션이 기준이다.

## 먼저 읽을 문서

1. `PROJECT.md`
2. `CURRENT.md`
3. `구현지시서.md` — 게임 구현의 단일 기준 문서 (요구사항 + 아트 디렉션 + 완료 기준)
4. `SOURCES.md`

## 이 프로젝트에서 지킬 기본 원칙

- 작업 범위는 현재 프로젝트 루트 기준으로 좁힌다.
- 게임 코드는 `app\` 폴더 안에서만 작업한다 (Vite + React + TypeScript).
- 구현 요구사항이 애매하면 임의로 정하지 말고 `구현지시서.md`를 우선 확인한다.
- `CURRENT.md`는 중요한 완료, 결정, 방향 변경이 생길 때마다 갱신한다.
- 계획과 달라진 구현은 `CURRENT.md`의 Deviations 항목에 이유와 함께 기록한다.
- 에셋은 CC0(저작권 표기 불필요)만 사용하고, 출처를 `SOURCES.md`에 기록한다.

## 메모

- 이 파일은 얇은 라우터 역할만 한다.
- 상세 정책은 workspace 루트의 `AGENTS.md`, `운영가이드.md`, 관련 workflow 문서에서 본다.
